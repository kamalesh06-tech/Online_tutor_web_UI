# Online_tutor_web_UI
Online Tutoring Web UI created for an AI prompt-based competition conducted by the AI &amp; ML Department at Mahendra Institute of Technology. Focuses on clean design, smooth student–tutor interaction, and modern UI principles
This project is a fully HTML-based online tutoring web user interface developed for an AI prompt–based competition conducted by the AI & ML Department at Mahendra Institute of Technology. The webpage is designed to represent a simple yet effective digital learning platform.

The UI includes structured sections such as a homepage overview, course offerings, tutor profiles, schedules, and contact information. It focuses on clarity, accessibility, and user-friendly navigation to ensure a smooth experience for students and tutors.

The design follows modern web layout principles using semantic HTML elements, ensuring responsiveness and easy customization. This project demonstrates how prompt-driven ideas can be transformed into a functional educational web interface using only HTML.

The project serves as a prototype for online education platforms and highlights the role of AI-assisted creativity in web UI development.
